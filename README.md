# A Nostalgic Friend Book

Project made for a Javascript Course, we were free to choose whatever project we wanted and I ended up with a nostalgic Friend Book. The book starts with two existing furry friends that I fetched from my own server/API using Node.js and Express.js. The user can then add a new friend or remove a friend from the book. 

I made the app using:
* React
* Redux
* Express.js
* Node.js
